## Описание

Данный репозиторий содержит тестовое задание.  
Проект подготовлен для быстрого и воспроизводимого запуска в изолированном окружении с использованием Docker.  
Все основные команды для работы с проектом вынесены в `Makefile`.

---

## Требования

Для запуска проекта на локальной машине необходимо наличие следующих инструментов:

- Docker  
- Docker Compose  
- Make  

---

### Инструкция по запуску

1. Откройте терминал и перейдите в корневую директорию проекта:
   ```bash
   cd test-task
2. Выполните сборку проекта:
   ```bash
   make build
3. Выполните запуск проекта:
   ```bash
   make up
